■ SQL
SQL　データベースを操作するための言語
DB2　BM社が提供するリレーショナル型データベース管理システム（RDBMS）
MySQL　Oracleが開発したデータベース（無料）
Maria DB　MｙSQLから派生した（開発者独立）
Oracle DB　Oracleが開発したデータベース
PostgreSQL オープンソースのRDBMS
SQLite オープンソースで軽量のRDBMS
SQL Server Microsoft社が開発したRDBMS
CRUD 「Create（生成）」「Read（読み取り）」「Update（更新）」「Delete（削除）」の頭文字 クラッド
トランザクション　複数の処理を一つにまとめたもの
コミット　関連する処理がすべて成功し、結果を確定させること
ロールバック　以前に正常に稼働していたある時点の状態に戻して復旧を試みること
排他制御（排他ロック）　共有資源に対して同時にアクセスしても問題なく動作できる制御
楽観ロック　同時アクセスはあまり起きないという楽観的な考え方の排他制御
悲観ロック　更新対象データを取得時にロックして他者が操作できないようにする方式
実行計画　ユーザが発行した問い合わせに基づきデータベース管理システム (DBMS) が内部的に生成する情報
チューニング　パフォーマンスの向上を図ること
パフォーマンス　システムなどの処理性能や実行速度
スロークエリ　データベース管理システム（DBMS）に対する問い合わせ（クエリ）のうち、一定の基準に照らして遅い
クエリログ　クライアントから実行された全ての SQL クエリを出力してくれるログのこと
IDE（統合開発環境）　ソフトウェア開発に必要なソフトウェアを一つに組み合わせ、同じ操作画面から統一的な操作法で利用できるようにしたソフトウェアパッケージ
Eclipse　様々なプログラミング言語によるソフトウェア開発に対応したオープンソースのIDE
NetBeans プロジェクト管理機能や入力支援機能を備えたオープンソースのIDE
SpringToolSuite Eclipseをベースにして、Spring開発に必要なものを組み込んだパッケージ
DBツール　データ管理を効率化するためのツール
A5:SQL Mk-2　データベース開発において生産性を向上させるべく開発されたSQL開発ツール
pgAdmin PostgreSQLの管理ツールの一つで、ネットワークを通じて遠隔のデータベースサーバをグラフィカルな表示・操作体系（GUI）で管理することができるもの オープンソース
DBeaver　ぼ全てのDBに対応したSQLクライアントツール
MySQL Workbench　データ・モデリング、SQL 開発、およびサーバー設定、ユーザー管理、バックアップなどの包括的な管理ツール
Sequel Pro　データベースをExcelのような表で確認・編集をすることができるMac専用のアプリ
TeamSQL　DBをGUIで管理する為のソフトウェア
